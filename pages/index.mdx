# Express.js How to make your own webapp

This guide will walk you through how to build your own webapp starting from what is serverside and nodejs to Authorization in express.

## Technologies you will learn throughout this course

- Node.js
- Express.js
- Requests in web services
- Error Codes
- JSON
- MongoDB
- Mongoose.js
- Sessions
- Token Auth using JWT


## Pre-Requisites

- Working knowledge of HTML
- Basics of javascript

## Setting up your development environment

1. Install vscode or preferred Development environment.
    - [Visual Studio Code](https://code.visualstudio.com)
    - [Sublime Text](https://sublimetext.com)
    - [Atom](https://atom.io)

2. Install Node.js on the current platform you're on.
    - Windows - [Node.js](https://nodejs.org/en/)
    - MacOS x64 - [Node.js](https://nodejs.org/en/)

    - MacOS M1
        - Install [homebrew](https://brew.sh)
        - Run `brew install nvm` in terminal
        - Run `nvm install node`

    - Linux
        - Follow the guide for your distro on https://nodejs.org/en/download/package-manager/
